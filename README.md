# 🎯 数当てゲーム - 知育アプリ

1〜100の数字を当てる知育ゲーム。質問力を育てながら楽しく遊べるPWAアプリです。

## 🎮 ゲーム概要

### 基本ルール
- **目的**: 1〜100の中から正解の数字を当てる
- **ヒント**: 何回でも質問できる（無制限）
- **解答権**: 5回まで
- **勝利条件**: 解答権を使い切る前に正解する

### 対象年齢
- **メインターゲット**: 幼稚園児〜小学校低学年
- **サブターゲット**: 高齢者、大人も楽しめる設計

## 💡 主な機能

### ヒントシステム
1. **自由入力ヒント**
   - 好きな数字を入力して「より大きい？」「より小さい？」で質問
   - 戦略的な二分探索が可能

2. **固定質問ヒント**
   - 偶数？ / 奇数？
   - 5の倍数？
   - 1〜50の間？

3. **複合条件絞り込み**
   - 全ての質問結果を組み合わせて候補を絞り込み
   - 例: 「1〜50」かつ「5の倍数」かつ「偶数」= 5個 (10,20,30,40,50)

### 教育的要素
- リアルタイムで候補数を表示
- ヒント使用回数のカウント
- クリア時の評価コメント
  - 7回以下: 「素晴らしい質問力です！🌟」
  - 8-10回: 「よく考えて当てられましたね！👏」
  - 11回以上: 「次はもっと少ないヒントで...💪」

## 🚀 セットアップ

### 必要な環境
- Node.js 14以上
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/your-username/number-guess-game.git
cd number-guess-game

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm start
```

### ビルド

```bash
# 本番用ビルド
npm run build

# ビルドファイルは build/ ディレクトリに生成されます
```

## 📱 PWAとしてインストール

### スマホにインストール

#### iOS (Safari)
1. Safariでアプリを開く
2. 共有ボタンをタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

#### Android (Chrome)
1. Chromeでアプリを開く
2. メニュー（︙）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

### デスクトップにインストール

#### Chrome / Edge
1. アドレスバー右側の「インストール」アイコンをクリック
2. 「インストール」をクリック
